# 智能写作助手 (Writing Assistant)

## 项目简介

基于 Gemini 的智能写作助手，支持多种写作模式和风格定制。
这是 Phase 2（提示工程）的实战项目，综合运用了课程中学到的：

- Prompt 结构设计
- System Prompt 角色设定
- Few-shot 示例
- 输出格式控制

## 功能特性

- ✍️ **多种写作模式**：博客、邮件、文案、代码文档
- 🎨 **风格定制**：正式/轻松、简洁/详细
- ⚡ **流式输出**：实时显示生成内容
- 📝 **续写改写**：基于已有内容优化
- 💾 **保存导出**：保存生成的内容

## 技术栈

- Python 3.10+
- Google Generative AI (Gemini)
- Rich (终端美化)

## 快速开始

### 1. 安装依赖

```bash
cd projects/phase-02-writing-assistant
pip install -r requirements.txt
```

### 2. 配置环境

确保项目根目录的 `.env` 文件包含：

```
GOOGLE_API_KEY=your-gemini-api-key
```

### 3. 运行

```bash
python main.py
```

## 学习要点

1. **Prompt 工程**：如何设计高质量的提示词
2. **角色设定**：使用 System Prompt 定义 AI 身份

4. **风格控制**：通过参数调整输出风格
