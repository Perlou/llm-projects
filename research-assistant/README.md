# 自动化研究助手

## 项目简介

构建一个能够自动搜索、阅读、总结文献的研究助手 Agent，支持网络搜索、文档阅读、笔记整理和报告生成。

## 功能特性

- ✅ ReAct Agent 架构（思考-行动-观察循环）
- ✅ 多工具协作（搜索、阅读、写作）
- ✅ 网络信息检索
- ✅ 文献阅读和总结
- ✅ 笔记自动整理
- ✅ 研究报告生成
- ✅ 历史记忆管理

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 填入 API Keys
```

### 3. 运行研究助手

```bash
python main.py
```

## 核心架构

### ReAct Agent 循环

```
┌──────────────────────────────────────────────┐
│                 用户任务                      │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  🤔 思考 (Thought)                           │
│  分析当前状态，决定下一步行动                │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  🔧 行动 (Action)                            │
│  选择并调用合适的工具                        │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  👁️ 观察 (Observation)                       │
│  获取工具执行结果                            │
└──────────────────────────────────────────────┘
                    │
                    ▼
              任务完成？
              /      \
           否 /        \ 是
             ▼          ▼
         继续循环    输出结果
```

### 可用工具

| 工具           | 功能     | 参数           |
| -------------- | -------- | -------------- |
| `search`       | 网络搜索 | query          |
| `read_url`     | 阅读网页 | url            |
| `take_note`    | 记录笔记 | title, content |
| `list_notes`   | 查看笔记 | -              |
| `write_report` | 生成报告 | title          |

## 技术栈

- Python 3.10+
